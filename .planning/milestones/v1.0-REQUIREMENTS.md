# Requirements Archive: v1.0 MVP

**Archived:** 2026-02-19
**Milestone:** v1.0 MVP

## v1 Requirements - Complete

### Timer

- [x] **TMR-01**: Timer displays time in MM:SS format
- [x] **TMR-02**: Timer supports three modes: Focus (25:00), Short Break (5:00), Long Break (15:00)
- [x] **TMR-03**: After 4 focus sessions, auto-select Long Break; otherwise Short Break
- [x] **TMR-04**: Start control begins countdown
- [x] **TMR-05**: Pause control stops countdown temporarily
- [x] **TMR-06**: Resume control continues from paused time
- [x] **TMR-07**: Skip control ends current session early and moves to next
- [x] **TMR-08**: Reset control returns to initial duration for current mode
- [x] **TMR-09**: Display shows current mode (Focus/Short Break/Long Break)
- [x] **TMR-10**: Display shows session count (e.g., "Focus #2")
- [x] **TMR-11**: Auto-start toggle option (off by default) starts next session automatically
- [x] **TMR-12**: Timer persists running state across page refresh (store lastTick timestamp)

### Notifications

- [x] **NOTF-01**: Play audible beep when session ends
- [x] **NOTF-02**: Request browser notification permission on first interaction
- [x] **NOTF-03**: Send browser notification when session ends (if permitted)

### Session Notes

- [x] **NOTE-01**: Text input available during Focus sessions only
- [x] **NOTE-02**: Notes autosave while timer runs (debounced, 500ms)
- [x] **NOTE-03**: Note maximum length: 2000 characters
- [x] **NOTE-04**: Tags input allows comma-separated tags
- [x] **NOTE-05**: Maximum 10 tags, each max 20 characters, alphanumeric + dash only

### Session Records

- [x] **SESS-01**: On Focus session end (timer hits 0 or Skip), save session record
- [x] **SESS-02**: Session record includes: id (uuid), start timestamp, end timestamp
- [x] **SESS-03**: Session record includes: planned duration seconds, actual duration seconds
- [x] **SESS-04**: Session record includes: mode (Focus only for history)
- [x] **SESS-05**: Session record includes: note text, tags array

### History

- [x] **HIST-01**: History displays list of Focus session records, newest first
- [x] **HIST-02**: Each item shows: date/time, actual duration, note preview (truncated), tags
- [x] **HIST-03**: Click item opens details drawer/modal
- [x] **HIST-04**: Details show: full note, start/end timestamps, duration
- [x] **HIST-05**: Details allow editing note and tags
- [x] **HIST-06**: Details allow deleting record
- [x] **HIST-07**: Filter by date range: Today, Last 7 days, Last 30 days, All
- [x] **HIST-08**: Search input filters by text in notes and tags

### Stats

- [x] **STAT-01**: Display total focus time today
- [x] **STAT-02**: Display total focus time last 7 days
- [x] **STAT-03**: Display number of focus sessions today
- [x] **STAT-04**: Display longest focus session in selected range

### Keyboard Shortcuts

- [x] **KEY-01**: Space key toggles Start/Pause/Resume
- [x] **KEY-02**: Enter key saves note when focus is on note field (prevent form submit)
- [x] **KEY-03**: Cmd/Ctrl+K focuses search box in history

### Navigation

- [x] **NAV-01**: Tab-based navigation: Timer | History | Stats
- [x] **NAV-02**: Active tab visually indicated

### Data Persistence

- [x] **DATA-01**: IndexedDB schema with version number
- [x] **DATA-02**: Migration stub for future schema changes
- [x] **DATA-03**: Settings stored: durations, auto-start preference
