## User Intent

Refine the timer view to match the provided mockup, focusing on:
1. Full-height split panel layout
2. Repositioned daily goal indicator (bottom left)

### Top Bar (New)
- Left: Status indicator dot + "Focus Session Active" text
- Right: Notification bell icon + User avatar
- Remove current Help/Settings buttons (relocate to sidebar or keep as-is)

### Timer Section (Left Pane)
- DEEP WORK badge (blue pill)
- Circular progress ring with gaps (4 arc segments)
- Large time display "24:59"

- **Horizontal button row:** Reset (left) | Play/Pause (center, larger) | Skip (right)
- **Daily Goal card** (bottom left of timer pane):
  - "DAILY GOAL" label
  - "4/8 Sessions" text
  - Progress bar
  - "50%" percentage

### Active Session Panel (Right Pane) - Full Height
- **Header:**
  - "Active Session" title
  - "Configure your current focus task" subtitle

- **CURRENT TASK section:**
  - Input field with icon
  - Placeholder: "What are you working on?"

- **SESSION NOTES section:**
  - Formatting toolbar: Bold (B), List, Link icons
  - Textarea for notes
  - Placeholder: "Brainstorming key UI components..."

- **TAGS section:**
  - Tag pills with # prefix (e.g., #Work, #Design)

## Decisions

### Locked (User Requirements)
- Full-height split panel layout
- Daily goal card positioned bottom-left of timer pane
- Complete Session button at bottom of right panel
- Horizontal button layout (reset | play | skip)
- Top bar with session status indicator

### Claude's Discretion
- Exact spacing and sizing values
- Responsive behavior at smaller breakpoints
- Animation/transition details
- Icon choices (use existing Lucide-style inline SVGs)

### Deferred
- Notification bell functionality
- User avatar/profile features
- Actual rich text formatting (toolbar can be UI-only for now)
